Commandes bot serveur des gameurs

Help Command

This command displays the help information for all available commands.

Usage:
!help

filmprovider 🔍

Obtenez les fournisseurs de location et d'achat de films dans une région spécifique (FR, BE ou CH) et affichez les détails du film.

Utilisation: !filmProvider <region> <nomFilm>. Exemple: !filmProvider FR Inception

film 🎥

Recherchez un film par nom et affichez ses détails.

Utilisation: !film <nomFilm>. Exemple: !film Inception

avenir 🔮

Affichez une liste de films à venir.

Utilisation: !aVenir

Liste de toutes les commandes !

Usage: !commandes

nasaimagedujour 🖼️

Obtenez l'image du jour de la NASA.

Utilisation

!nasaImageDuJour

lancerjeu 🕹️

Lance un jeu par son nom dans la base de données (accessible uniquement pour l'administrateur : Loganner pour le moment) Usage: !lancerJeu nomdujeu

ajouterjeu ➕

Ajoute un jeu dans la base de données. Usage: !ajouterJeu nomdujeu iddujeu (admin)

mettreajourjeu 🔄

Mise à jour d'un jeu (admin). Usage: !mettreAJourJeu anciennom nouveaunom iddujeu

joueursjeuxfavoriscommuns 🎲

Obtenez la liste des joueurs ayant des jeux favoris en commun. Usage: !joueursJeuxFavorisCommuns

supprimerjeu ❌

Pour supprimer un jeu de la base de données (admin). Usage: !supprimerJeu nomdujeu

listejeux 📋

Affiche tous les jeux de la base de données (admin). Usage: !listeJeux
